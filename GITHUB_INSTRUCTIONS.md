# Como enviar para o GitHub

O repositório Git local já foi inicializado e todos os arquivos foram commitados.

Para enviar este projeto para o seu GitHub, siga os passos abaixo:

1.  **Autentique-se e Envie**
    Como já configurei o link do repositório para você, basta rodar o comando abaixo no terminal:

    ```bash
    git push -u origin main
    ```

    O terminal pedirá:
    *   **Username**: `vendoideias2`
    *   **Password**: Seu **Personal Access Token** (não a senha de login).

    > Se você ainda não tem um Token, gere um em: [Settings > Developer Settings > Personal Access Tokens](https://github.com/settings/tokens).

2.  **Pronto!**
    Seu código estará em: https://github.com/vendoideias2/crm-imoveis
